# astrbot-plugin-everyday-weatherforecast

自用，勿下载，勿下载

一个专门用于QQ群聊每日播报天气的插件

本插件基于[心知天气](https://www.seniverse.com)生成天气播报，请先在插件配置输入和风api和默认所在城市，api需要使用专业版，建议先使用试用版了解情况看是否满意再购入专业版。输入api后可通过/weather来检验是否成功，可在配置中选择是否自动播报以及自动播报时间以及希望播报的群聊
